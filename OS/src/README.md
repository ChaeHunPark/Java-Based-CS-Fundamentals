## 💡 핵심 용어 정리

| 용어 | 설명 | 비유 |
| :--- | :--- | :--- |
| **Process** | 실행 중인 프로그램 (메모리 점유) | 준비된 요리 세트 |
| **Thread** | 프로세스 내 실제 작업의 흐름 | 요리사의 손 |
| **RAM** | CPU가 즉각 접근하는 작업 공간 | 요리용 도마 |
| **PID** | 프로세스 고유 식별 번호 | 프로세스의 주민번호 |
| **Multi-Threading** | 하나의 프로세스 자원을 공유하며 병렬 작업 | 요리사가 양손을 쓰는 것 |
| **Context Switching** | CPU가 작업 전환 시 현재 상태를 저장하고 복구하는 과정 | 요리하던 중간 단계를 기억해두기 |
| **Scheduling** | 어떤 프로세스/스레드에게 CPU를 줄지 결정하는 정책 | 주방장의 요리 순서 결정 |
| **Time Sharing** | CPU 시간을 잘게 나누어 여러 작업에 할당하는 방식 | 요리사가 여러 화구를 번갈아 확인 |
| **Preemptive** | OS가 강제로 CPU를 뺏어 다른 작업에 줄 수 있음 | 주방장이 요리사 손을 강제로 옮김 |
| **Round Robin** | 정해진 시간(Time Slice)만큼 돌아가며 CPU 할당 | 모든 요리를 1분씩 번갈아 하기 |
| **Priority** | 특정 작업에 더 많은 CPU 기회를 주는 수치 | VIP 주문 먼저 처리하기 (힌트) |
