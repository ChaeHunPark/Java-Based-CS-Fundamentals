## 📝 학습 리스트
- [x] **Step 01: OSI 7계층 & TCP 소켓 통신** [코드](./Step01_Socket.java)
    - **내용**: OSI 7계층 이론 학습 및 Java `ServerSocket` / `Socket`을 이용한 1:1 통신 구현.
    - **핵심 통찰**:
        1. **4계층(TCP)**: `Socket` 연결 시 내부적으로 **3-Way Handshake**가 발생하여 신뢰성 있는 통신 채널을 형성함.
        2. **캡슐화**: 우리가 보낸 "Hello" 문자열이 각 계층을 지나며 헤더가 붙어 패킷/프레임으로 변하는 과정 이해.
  ### 🔍 핵심 통찰: TCP 3-Way Handshake
| 단계 | 플래그 | 의미 | 비유 |
|:---:|:---:|:---|:---|
| 1 | **SYN** | 클라이언트 -> 서버 연결 요청 | "저기요, 대화 가능하신가요?" |
| 2 | **SYN-ACK** | 서버 -> 클라이언트 응답 및 요청 | "네, 들립니다. 제 말도 들리세요?" |
| 3 | **ACK** | 클라이언트 -> 서버 최종 응답 | "네, 잘 들립니다. 말씀하시죠!" |

> **주의**: 이 과정은 애플리케이션(Java)이 아닌 운영체제(OS)의 TCP 스택에서 자동으로 수행됩니다. `accept()`가 리턴되었다면 이 과정이 이미 성공적으로 끝났음을 의미합니다.
    - **실습**: 로컬 환경에서 서버-클라이언트 간 메시지 송수신 성공.

## 💡 네트워크 핵심 용어

| 용어 | 설명 | 비유 |
| :--- | :--- | :--- |
| **IP (Internet Protocol)** | 컴퓨터의 고유 주소 (3계층) | 택배를 보낼 때의 집 주소 |
| **Port** | 한 컴퓨터 내의 프로세스 식별 번호 (4계층) | 아파트 호수 (8080호, 3306호 등) |
| **TCP** | 연결 지향형, 신뢰성 보장 (분실 시 재전송) | 전화 통화 (상대방이 받았는지 확인하며 대화) |
| **UDP** | 비연결형, 속도 우선 (분실 감수) | 방송/유튜브 스트리밍 (상대방이 보든 말든 일단 송출) |
| **Handshake** | 데이터를 주고받기 전 연결을 확립하는 과정 | 대화 시작 전 "저기요?", "네?", "드릴 말씀이 있어서요" 하는 절차 |
