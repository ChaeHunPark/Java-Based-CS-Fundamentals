## 🎯 학습 철학 (Philosophy)
1. **First Principles:** 근본적인 동작 원리에 집중한다.
2. **Evidence-Based:** 모든 이론은 Java 코드로 구현하여 증명한다.
3. **Layered Learning:** 메인 README는 가이드를, 폴더별 README는 상세 노트를 담당한다.
